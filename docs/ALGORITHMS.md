# Algorithms

В библиотеке используются классические алгоритмы для разбора и обработки арифметических выражений.

---

## 1. Алгоритм Shunting-yard

Используется для преобразования инфиксной записи (обычной записи выражений) в постфиксную (RPN).

### Основная идея

- используется стек операторов
- учитываются приоритеты операций
- обрабатываются скобки

### Пример

Вход:

```c

3 + 4 * 2 / (1 - 5)

```

Выход (RPN):

```c

3 4 2 * 1 5 - / +

```

---

## 2. Вычисление постфиксного выражения

### Основная идея

- используется стек значений
- при встрече числа — кладём в стек
- при встрече оператора:
  - берём последние операнды
  - выполняем операцию
  - кладём результат обратно

### Пример

```c

3 4 2 * +

```

Шаги:

- 3 → стек: [3]
- 4 → стек: [3, 4]
- 2 → стек: [3, 4, 2]
- * → стек: [3, 8]
- + → стек: [11]

## 3. AST (Abstract Syntax Tree)

AST — это дерево, представляющее структуру выражения.

### Структура

- листья — числа или переменные
- внутренние узлы — операции

### Пример

Выражение:

```c

3 + 4 * 2

```

Дерево:

```c

```
+
```

/ 
3   *
/ 
4   2

```

---

## 4. Вычисление AST

Вычисление происходит рекурсивно:

- если узел — число → вернуть значение
- если переменная → найти в таблице
- если операция:
  - вычислить левое поддерево
  - вычислить правое
  - применить оператор

---

## 5. Упрощение выражений

Применяются правила:

- x + 0 → x
- x * 1 → x
- x * 0 → 0
- константные подвыражения вычисляются

### Пример

```c

2 + 3 → 5
x + 0 → x

```

---

## 6. Дифференцирование

Используются базовые правила:

- d/dx x = 1
- d/dx c = 0
- d/dx (f + g) = f' + g'
- d/dx (f * g) = f*g' + g*f'
- d/dx (f / g) = (f'*g - f*g') / g^2

### Пример

```c

x^2 + 2*x

```

Результат:

```c

2*x + 2

````
